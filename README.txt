IoT Temperature Collection Project

Vishnu Jaganathan

Using Microsoft's node_modules for getting data from the pins of a BME 280 sensor. This program modifies the data acquisition process to suit a BMP 280 on a Raspberry Pi
and creates a way for the data to be sent to the Azure IoT Cloud.

call program with 'sudo node index.js [connection string]' on a Raspberry Pi Model 3B
